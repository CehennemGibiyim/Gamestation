====================================================
  LUX MALLAR KAR HESAPLAMA - KURULUM REHBERI
====================================================

SECNEK 1: Direk EXE Yapmak (ONERILEN)
----------------------------------------
1. BUILD.bat dosyasina cift tiklayin
2. Otomatik olarak:
   - Python kontrolu yapar
   - PyInstaller kurar
   - EXE dosyasini olusturur
3. dist/ klasorundeki LuxMallarKarHesaplama.exe
   dosyasini istediginiz yere tasiyin
4. Artik Python kurulu olmayan bilgisayarlarda
   da calisir!

NOT: BUILD.bat icin Python kurulu olmasi gerekir.
Python: https://www.python.org/downloads/
Kurulumda "Add Python to PATH" MUTLAKA isaretleyin!

SECNEK 2: Python ile Direk Calistirmak
----------------------------------------
1. Python kurulu ise CALISTIR.bat'a tiklayin
   VEYA
   komut satirinda: python lux_kar_hesaplama.py

====================================================
  AYARLAR
====================================================
Program ayarlarini lux_settings.json dosyasinda
saklar. EXE ile ayni klasorde olmalidir.
